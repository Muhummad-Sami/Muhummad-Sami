## 🎓 Academic Projects

### 🌐 Hotel Network Design — Cisco Packet Tracer
A complete enterprise-level hotel network simulation designed and configured using Cisco Packet Tracer.
- **Tool:** Cisco Packet Tracer
- **Scenario:** Multi-floor hotel with dedicated VLANs per department — Reception, Rooms, Restaurant, Conference Hall, Management, and IT
- **Topology:** Hierarchical three-layer model (Core, Distribution, Access)
- **Features & Configurations:**
  - VLAN segmentation for each department to isolate traffic
  - Inter-VLAN routing via Router-on-a-Stick (subinterfaces)
  - DHCP server configured on router to auto-assign IPs per VLAN
  - Static routing and default gateway configuration
  - Wireless Access Points configured for guest Wi-Fi (Rooms & Restaurant VLANs)
  - DNS & HTTP server simulation for internal hotel portal
  - SSH configured on all routers and switches for secure remote management
  - Port security on access layer switches to restrict unauthorized devices
  - Spanning Tree Protocol (STP) enabled to prevent network loops
  - End-to-end connectivity verified with ping and traceroute tests
